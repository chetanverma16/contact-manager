# Contact Manager For Work Club HQ


### Made with AWS Amplify (Hosting + Cognito + AppSync)

## How To Setup Locally
 - Clione this repo
 - `git checkout amplify`
 - run `npm install` or `yarn install`
 - run `npm start` ot `yarn start`
 
 
 ## Features
  - `Login + Sign Up` (Cognito)
  - Multiple Themes 
  - Add Contact
  - Edit Contact
  - Delete Contact
  - GraphQL (AWS AppSync)
